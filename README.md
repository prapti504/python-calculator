# python-calculator

Basic calculator built using Python functions, loops, and conditional statements.

## Features

* Addition
* Subtraction
* Multiplication
* Division

## How to Run

1. Clone the repository

```bash
git clone https://github.com/prapti504/python-calculator.git
```

2. Open the project folder

```bash
cd python-calculator
```

3. Run the Python file

```bash
python calculator2.py
```
